# 英语学习应用 (English Learning App) - UI 视觉与交互设计规范 (面向中国用户)

本设计规范专为**中国英语学习者**的审美与使用习惯定制。摒弃了过于科幻、AI 化和纯英文的极客风格，转而采用**温润极简、高易读性、结构清晰**的本土化设计路线。整体对标国内一线主流学习应用（如：不背单词、墨墨背单词、扇贝英语）。

---

## 1. 视觉设计系统 (Design System)

### 1.1 色彩体系 (Color Palette)
为营造轻松、专注且具有学术成长感的学习氛围，主色调选用具有生命力的护眼绿与温润蓝：

| 色样 | 十六进制 | 角色 | 视觉释义 |
| :--- | :--- | :--- | :--- |
| **生机绿 (Growth Green)** | `#2B8A3E` 至 `#40C057` | 主色/正解/完成度 | 象征知识的成长、平静专注、正确答题的反馈。 |
| **温润蓝 (Warm Teal)** | `#0C8599` 至 `#15AABF` | 辅助色/专项训练 | 用于各个单项练习卡片，带来严谨和逻辑感。 |
| **活力橙 (Energy Orange)** | `#F76707` 至 `#FF922B` | 警示/打卡火苗/金币 | 刺激感官，用于连续打卡火苗 🔥、复习提醒和成就激励。 |
| **雪白底 (Ice White)** | `#F8F9FA` | 背景底色 (Light) | 默认主背景，柔和护眼，避免纯白刺眼。 |
| **石墨灰 (Graphite Slate)**| `#212529` / `#495057` | 标题/正文文本 | 提供高对比度的清晰阅读体验，拒绝低对比度的刺眼设计。 |

### 1.2 字体与版式 (Typography)
*   **中文字体**: 优先使用 `PingFang SC`（苹方）、`Microsoft YaHei`（微软雅黑）以及系统默认的 `sans-serif`。
*   **英文字体**: 数字与英文单词使用 `Inter` 或 `Plus Jakarta Sans`，确保音标与字母清晰可辨。
*   **排版原则**: 
    *   **词不离句**: 界面上出现的英文单词必须配有中文释义，且大小对比分明。
    *   **高可读性**: 行高控制在 `1.5 - 1.6`，字间距适中，卡片之间留白充裕，防止视觉疲劳。

### 1.3 容器质感 (Texture & Elevation)
*   **去 AI 浮夸化**: 放弃大面积深色霓虹渐变与炫目的毛玻璃投影。
*   **扁平质感卡片**: 卡片背景采用纯白（`#FFFFFF`），搭配极淡的边框（`1px solid #E9ECEF`）与非常微弱的弥散投影（`box-shadow: 0 4px 16px rgba(0, 0, 0, 0.015)`）。
*   **大圆角规范**: 容器圆角统一为 `16px`，操作按钮圆角为 `24px` 或全圆角（Pill），带来亲和力与现代感。

## 2. 首页布局设计 (每日学习 Tab)

面向中国用户的界面，必须有直观的**中文引导**、**清晰的复习提示**以及**易于分享的打卡卡片**。

```text
+---------------------------------------------------------+
| [头像] 你好，Alex！                         🔥 连续打卡 7 天 |
| 学习目标：大学英语四级                                     |
+---------------------------------------------------------+
|                  今日学习任务舱                          |
|  +---------------------------------------------------+  |
|  |                 [ 进度：50% ]                     |  |
|  |               今日目标：已完成 2/4                 |  |
|  |                                                   |  |
|  |  (1) 每日单词 [✔] --> (2) 每日听力 [✔]             |  |
|  |                                                   |  |
|  |  (3) 每日句子 [进行中] --> (4) 每日文章 [未开始]     |  |
|  |                                                   |  |
|  |              [  继续今日学习 (Resume)  ]           |  |
|  +---------------------------------------------------+  |
+---------------------------------------------------------+
|  ⚡ 温故知新：有 5 个生词需要复习，点击立即消灭！            |
+---------------------------------------------------------+
|  每日金句 (Daily Quote)                                 |
|  "The limits of my language mean..."                    |
|  语言的边界，就是世界的边界。                  [朗读图标] |
+---------------------------------------------------------+
```

### 2.1 头部区域 (Header)
*   **左侧**: 用户圆形头像，旁边展示中文欢迎词“你好，[姓名]！”以及当前的学习目标卡片（如“CET-4”、“雅思冲刺”）。
*   **右侧**: 打卡火苗 🔥，显示中文“连续打卡 N 天”。

### 2.2 今日学习任务舱 (Core Task Panel)
*   **配色**: 选用柔和的`生机绿渐变`（由 `#37B24D` 渐变至 `#2B8A3E`）。
*   **任务卡逻辑**: 中国用户习惯清晰的网格或步骤流线。我们将步骤设计为“田”字形或横向引导链：
    1.  **每日单词**（显示已背/总数，如：`15/20个`）
    2.  **每日听力**（显示状态：`已听完`）
    3.  **每日句子**（显示状态：`进行中`）
    4.  **每日文章**（显示状态：`未开始`）
*   **主动作按钮**: 白色大胶囊按钮，配绿色文字“继续今日学习”。

### 2.3 巩固复习提示 (Smart Review Alert)
*   位于任务舱下方，采用微带橘黄色的淡色警示框（背景 `#FFF9DB`，文字 `#F59F00`），直白提醒：“**⚡ 温故知新：还有 5 个单词需要复习，点击立即消灭！**”。

## 3. 底部导航栏与 Tab 页内容 (本土化设计)

底部导航栏为中国用户最熟悉的 **4 标签设计**，图标下方配有清晰的中文字标。

### 3.1 Tab 1: 每日学习 (学习首页)
*   核心任务舱、复习提示、每日中英双语金句（点击配真人原声朗读）。
*   快捷入口：“我的生词本”（展示生词数）、“错题收集”（展示错题数）、“打卡日历”。

### 3.2 Tab 2: 专项突破 (单项练习)
*   顶部横向滑动分类：“音标/字词根/词汇/语法/阅读/听力”。
*   下方展示对应的**专项训练书架**，比如：
    *   *音标纠音书卡*：带有发音口型指引和测验。
    *   *词根词缀书卡*：以树状思维导图展示核心词根。
    *   *词库选择卡*：提供国内常见的考试词库（中考、高考、四级、六级、考研、雅思）。

### 3.3 Tab 3: 学习榜单 (排行榜)
*   符合中国用户“熟人比拼、社群激励”的特征。
*   **我的排名**：横向浮动卡，展示我的排名、头像、累计能量值（LP）。
*   **段位头衔**：青铜先锋 $\rightarrow$ 白银极光 $\rightarrow$ 黄金词霸 $\rightarrow$ 英语大师（汉字呈现，更具归属感）。
*   **排行榜列表**：前三名展示金银铜牌。右侧有快捷的“点赞 👍”和“比心 ❤️”互动，用于打卡监督。

### 3.4 Tab 4: 个人中心 (我)
*   个人头像、学习目标修改。
*   **学习数据大盘（战报）**：累计学习天数、累计学习分钟、已掌握单词、已阅读文章。
*   **图表分析**：
    *   *学习时间折线图*（展示本周每日打卡时长）。
    *   *五维能力图*（词汇力、语法力、阅读力、听力、发音）。
*   **成就徽章陈列柜**：展示解锁的成就（如“百词斩”、“听觉大师”）。
*   **系统设置**：字体缩放、深色模式开关、推送提醒时间设置。
