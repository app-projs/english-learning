# 英语学习应用 - V3 深度学习与云端同步版本需求文档

## 1. 版本目标与范围

V3 版本是本应用的**完全体版本**。本版本聚焦于提升复习效率、精细化基础专项练习（音标与字词根），以及实现全场景的多设备同步和数据分析。核心实现艾宾浩斯智能算法驱动的全局学习计划，并通过细粒度的学习报告实现用户的学习反馈闭环。

---

## 2. 核心功能点定义

### 2.1 智能复习引擎 (SM-2 Algorithm Integration)
*   **艾宾浩斯复习日程分配**:
    *   在每日练习包中自动混入复习项。复习项根据用户上次答题的困难度评级（1-5级），使用 **SM-2 算法**动态计算下一次复习时间。
*   **消灭错题挑战**:
    *   错题集引入多重关卡机制。只有用户对某道错题连续 3 次答对，该题才会被自动标记为“已消灭”，否则继续处于高频复习队列。

### 2.2 基础专项精度打磨 (Phonetics & Word Roots)
*   **互动音标馆**:
    *   提供完备的 IPA 互动音标表。
    *   集成高清真人发音口型视频动画、舌位侧面解剖图。
    *   **AI 纠音仪**: 使用 AI 发音测评（如声母、韵母、声调级别定位），找出用户发错音的字母，并以高亮和音频提示形式指导修改。
*   **字词根思维导图**:
    *   将词根词缀百科图形化为“单词树”。用户可点击树枝展开相关同根词、前缀词、后缀词。
    *   **拆词匹配游戏**: 通过解谜游戏（将前缀、后缀、词根拖拽拼装成完整单词并与释义连线）强化构词逻辑记忆。

### 2.3 全场景多设备云端同步 (Cloud Sync & Database Merge)
*   **离线操作队列**:
    *   在离线或无网环境下，用户依然可背单词、读下载好的文章、保存做题历史。所有的修改和打卡记录被写入本地 SQLite 离线队列。
*   **智能冲突合并**:
    *   当网络恢复时，后台自动触发数据同步。
    *   采用逻辑时间戳合并冲突（如多设备打卡记录合并、生词本去重）。

### 2.4 数据统计与能力画像 (Study Reports & Radar Charts)
*   **五维能力雷达图**:
    *   对用户的做题历史进行特征提取，生成“词汇力、语法力、阅读力、听觉力、口语力”五维得分，直观展示用户英语的长短板。
*   **深度学习报告**:
    *   每周日生成“周学习报告”和“月度报告”，展示词汇增长曲线、精读文章数对比图、错题率走势。
    *   基于雷达图数据，AI 自动推荐下一周应加强的单项专项（如：“*检测到您的听力得分偏低，下周推荐增加听力专项时长*”）。
