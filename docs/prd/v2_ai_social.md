# 英语学习应用 - V2 AI 与社交突破版本需求文档

## 1. 版本目标与范围

V2 版本将应用由本地单机版升级为**网络化与智能化的在线英语学习平台**。重点突破“听力专项”与“口语练习”的内容建设，并通过“学习榜单”与“积分机制”注入社交动力。本阶段将引入服务端 API 交互，搭建用户账号体系与云端数据存储。

---

## 2. 核心功能点定义

### 2.1 用户系统与云端底座
*   **账号体系**:
    *   支持用户注册、手机/邮箱登录、第三方快捷登录。
    *   采用 JWT 鉴权管理网络请求会话。
*   **网络数据层 (Dio)**:
    *   构建网络请求层，提供请求拦截器、全局错误提示、数据缓存策略（优先读缓存，异步拉取新内容）。

### 2.2 每日听力与听力专项 (Listening Module)
*   **听力音频播放器**:
    *   集成本地或在线音频播放，提供波形图显示、开始/暂停/快进退。
    *   **变速播放功能**: 提供 `0.8x / 1.0x / 1.2x / 1.5x` 播放语速控制。
*   **听力测验玩法**:
    *   **盲听挑战**: 听完后完成 3 道阅读理解选择题。
    *   **精听听写 (Dictation)**: 挖空特定句子的核心词，用户听录音后拼写出完整单词，系统自动比对并进行拼写纠错。

### 2.3 学习榜单积分制 (Leaderboard System)
*   **能量值计算引擎 (LP Engine)**:
    *   定义 LP (Learning Points) 积分规则：打卡成功得 50 LP；答对单词得 2 LP；听力完成得 30 LP 等。
*   **多维度榜单**:
    *   **全球周排行榜**: 显示本周全球用户积分排名，每周一 0:00 重置。
    *   **好友竞技榜**: 展示微信/系统好友的排名与学习火苗。
*   **段位晋升机制**:
    *   根据累积 LP 分数，解锁 `青铜` $\rightarrow$ `白银` $\rightarrow$ `黄金` $\rightarrow$ `大师` 称号，并在头像框显示专属特效。

### 2.4 AI 服务初版集成 (AI & Cloud Services)
*   **划词翻译与一键收藏**:
    *   调用云端 AI 翻译 API，在长文阅读中支持双击取词/取句，即时获取中文释义和语法点解析。
*   **AI 口语陪练 (AI Talking Coach)**:
    *   **场景库**: 提供“旅行问路”、“餐厅点餐”、“酒店值机”等 10+ 互动场景。
    *   **智能对话**: 采用 LLM 驱动，AI 根据用户的语音或文字输入进行拟人化对答。
    *   **发音评分**: 用户可通过长按录音发送语音，AI 进行声学比对并提供流畅度、准确度打分。
