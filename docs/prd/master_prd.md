# 英语学习应用 (English Learning App) - 总产品需求文档 (Master PRD)

## 1. 产品概述与核心价值

### 1.1 产品愿景
本应用是一款专为英语学习者打造的**有阶段、有规律**的移动端学习软件。通过高度视觉化、本土化的设计，将每日学习任务串联成线性引导路径，配合垂直专项训练与艾宾浩斯智能复习算法，帮助用户在日常碎片时间中建立高频、持续、科学的语言学习习惯。

### 1.2 视觉设计风格定位 (温润极简风 - Warm Minimalism)
为给中国英语学习者带来友好、专注、解压的视觉体验，界面整体设计规范如下：
*   **色彩体系 (Palette)**:
    *   **主背景色 (Light Theme - 默认)**: 采用护眼暖白灰色（`#F8F9FA`），卡片采用纯白（`#FFFFFF`）背景。
    *   **生机绿 (主色 - Growth Green)**: 用于表示正确、进度及完成状态。采用 `#2B8A3E` 至 `#40C057` 的柔和渐变。
    *   **温润蓝 (辅助色 - Warm Teal)**: 用于单项练习、专项训练卡片，传递严谨感。采用 `#0C8599` 至 `#15AABF` 的渐变。
    *   **活力橙 (能量色 - Energy Orange)**: 用于打卡火苗 🔥、复习提醒和成就金币。采用 `#F76707` 至 `#FF922B` 的渐变。
    *   **石墨灰 (正文色 - Graphite Slate)**: 采用 `#212529`（重要标题）与 `#495057`（普通文本）提供清晰、高易读性的中英文文字对比。
*   **卡片与版式质感**: 
    *   容器圆角规范统一采用轻巧现代的 `16px`。
    *   卡片仅搭配极淡的描边（`1px solid #E9ECEF`）与微弱的弥散阴影（`box-shadow: 0 4px 16px rgba(0, 0, 0, 0.015)`），拒绝繁琐的荧光与重度毛玻璃。
*   **字体规范**: 
    *   中文字体：优先 `PingFang SC` 与微软雅黑。
    *   英文字体：数字与英文单词使用 `Inter` 或 `Plus Jakarta Sans`，确保音标与字母高度易读。

## 2. 首页设计 (每日学习 Tab) 详细定义

首页是用户每天打开 APP 的第一触点，其排版与逻辑需要聚焦于**减少选择负担，引导直接开始**。

```text
+---------------------------------------------------------+
| [头像] 你好，Alex！                         🔥 连续打卡 7 天 |
| 学习目标：大学英语四级                                     |
+---------------------------------------------------------+
|                  今日学习任务舱                          |
|  +---------------------------------------------------+  |
|  |                 [ 进度：50% ]                     |  |
|  |               今日目标：已完成 2/4                 |  |
|  |                                                   |  |
|  |  (1) 每日单词 [✔] --> (2) 每日听力 [✔]             |  |
|  |                                                   |  |
|  |  (3) 每日句子 [进行中] --> (4) 每日文章 [未开始]     |  |
|  |                                                   |  |
|  |              [  继续今日学习 (Resume)  ]           |  |
|  +---------------------------------------------------+  |
+---------------------------------------------------------+
|  ⚡ 温故知新：有 5 个生词需要复习，点击立即消灭！            |
+---------------------------------------------------------+
|  每日金句 (Daily Quote)                                 |
|  "The limits of my language mean..."                    |
|  语言的边界，就是世界的边界。                  [朗读图标] |
+---------------------------------------------------------+
```

### 2.1 页面结构与版面布局
1.  **顶部状态栏 (User Bar)**:
    *   **左侧**: 用户圆形头像，右侧并列展示个性化中文问候语（“你好，[用户名]!”）和当前的学习目标（如“大学英语四级” / “雅思冲刺”）。
    *   **右侧**: 橙色打卡火苗图标 🔥，显示连续打卡天数（如“连续打卡 7 天”）。
2.  **今日学习任务舱 (Core Task Panel) - 黄金焦点区**:
    *   **卡片背景**: 采用柔和的 `生机绿渐变`。
    *   **核心进度条**: 动态展示今日已完成的任务比（如 `已完成 2/4` 或 `50%`）。
    *   **步骤指示网格 (Step Nodes)**: 采用横向引导链或“田”字排布，标明学习闭环：
        *   (1) 每日单词（已背 15/20 个，显示已完成绿勾 [✔]）
        *   (2) 每日听力（显示已完成绿勾 [✔]）
        *   (3) 每日句子（高亮展示为 [进行中] 状态）
        *   (4) 每日文章（半透明展示为 [未开始] 状态）
    *   **主操作按钮**: 底部放置白色圆角大胶囊按钮，配绿色字体的“**继续今日学习**”。
3.  **智能巩固条 (Smart Review Banner)**:
    *   若今天有待复习的错题或生词，则在任务舱下方弹出一个温润的淡黄色警示条（背景 `#FFF9DB`，文字 `#F59F00`）：“*⚡ 温故知新：有 5 个生词需要复习，点击立即消灭！*”，点击直接跳转复习页面。
4.  **每日金句卡片 (Daily Quote Card)**:
    *   纯白微阴影卡片。
    *   展示精选英语名言、中文翻译。右下角配有一个扬声器图标，点击后触发真人原声或高品质 TTS 朗读。

## 3. 底部 Tab 导航及各页面内容定义

底部导航包含 4 个核心 Tab，图标下方配有清晰的中文字标。

### 3.1 Tab 1: 每日学习 (学习首页)
*   **核心定位**: 线性任务主导，引导每日规律化打卡。
*   **详细展示内容**:
    *   用户状态与打卡火苗。
    *   今日学习任务舱（动态步骤指示）。
    *   智能复习警示条。
    *   每日双语金句卡片。
    *   快捷工具网格：
        *   **我的生词本**（显示生词总数，如“34个单词”）。
        *   **错题收集**（显示错题总数，如“5道题目”）。
        *   **打卡日历**（显示本月已完成打钩的小日历图）。

### 3.2 Tab 2: 专项突破 (专项突破)
*   **核心定位**: 模块化自主训练，用于单项技能重点突破。
*   **页面展示布局**:
    *   顶部搜索框（可查找专项库中的文章、词汇、词根）。
    *   横向分类滑块：“音标 / 字词根 / 词汇 / 语法 / 阅读 / 听力”。
    *   **专项书架卡片列表**，采用温润蓝渐变图标与卡片：
        1.  **音标纠音 (Phonetics)**: 互动式音标表、口型图解、发音纠错测验。（显示“已学习 12/48”）。
        2.  **字词根记忆 (Word Roots)**: 词根词缀库、词根树状思维导图、拆词连线匹配游戏。（显示“已解锁 35 个词根”）。
        3.  **词汇积累 (Vocabulary)**: 选择中高考、四六级、考研、雅思等词库。（显示进度“四级: 1,200/4,500”）。
        4.  **句子语法 (Sentences)**: 语法专题精讲、句子排序、遮挡填空练习。（显示“已通关 8 个语法点”）。
        5.  **长文精读 (Reading)**: 分难度精读文章、长难句结构拆解、课后阅读理解题。（显示“已精读 18 篇文章”）。
        6.  **精听磨耳朵 (Listening)**: 日常/商务听力播放器（变速支持）、逐字 Dictation 听写测验。（显示“累计精听 12.5 小时”）。

### 3.3 Tab 3: 学习榜单 (学习榜单)
*   **核心定位**: 社交激励与段位攀升。
*   **页面展示布局**:
    *   **顶部 - 用户个人排行展示卡**:
        *   显示用户当前的全国/好友排名、头像、昵称、累积能量值。
        *   展示中文段位名称，如：`青铜先锋`、`白银极光`、`黄金词霸`、`英语大师`。
    *   **榜单列表 (Top 50)**:
        *   前三名分别带有金🥇、银🥈、铜🥉动态徽章。
        *   每个条目包含：名次、头像、昵称、本周获得的能量值、连续打卡天数。
        *   条目右侧配有快速互动按钮，用户可点击“点赞 👍”和“比心 ❤️”互送鼓励。
    *   **榜单切换**: 可一键在“全球排行榜”和“好友排行榜”之间无缝切换。

### 3.4 Tab 4: 个人中心 (我)
*   **核心定位**: 学习成果可视化与应用全局设置。
*   **页面展示布局**:
    *   **顶部个人信息**: 展示圆形头像、昵称、学习目标。
    *   **四大成就数据大盘（战报）**:
        *   `累计打卡 (天)` | `掌握单词 (个)`
        *   `精读文章 (篇)` | `精听时长 (小时)`
    *   **五维能力雷达图**:
        *   展示词汇力、语法力、阅读力、听力、发音的综合能力评估得分，直观呈现偏科现状。
    *   **周/月学习曲线图**:
        *   以柔和平滑的折线或柱状图展示近一月每天的学习时间分布。
    *   **成就徽章柜**:
        *   横向滑动的徽章墙。已解锁的徽章高亮显示，未解锁的置灰。
    *   **设置项**:
        *   修改每日学习目标。
        *   深色模式开关。
        *   提醒推送时间设置。
        *   本地缓存清理与云端备份。

## 4. 后续演进版本设计 (规划)

本 PRD 作为总纲，为后续快速迭代奠定基础：
*   **V1 版本 (MVP - 基础框架)**: 本地单机运行，核心跑通每日引导路径（单词卡片/测试、句子重排/翻译/填空、文章双语阅读）以及基于 SharedPreferences 和 SQLite 的本地错题本、主题切换及个人中心指标。数据采用 Mock 驱动。
*   **V2 版本 (AI 与社交突破)**: 引入用户系统与服务器端；上线学习榜单和积分计算；增加听力专项与音频播放器；集成 AI 服务（划词翻译接口、AI口语陪练初版）。
*   **V3 版本 (深度个性化与云同步)**: 实现全面的 SM-2 记忆算法复习日程表；增加音标与字词根的互动模块；实现完整的云端数据多设备同步、深度学习报告雷达图及 AI 自动口语纠错系统。
