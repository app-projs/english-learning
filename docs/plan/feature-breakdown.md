# 功能特性分解 (Feature Breakdown)

本篇文档对照最新的 [PRD.md](file:///d:/workspace/test/english-learning/docs/PRD.md)，对系统的各个模块进行底层功能的详细拆解，并设定开发状态与优先级。

---

## 🚀 1. 每日练习核心模块 (Daily Practice)

每日任务引擎是驱动用户规律性学习的核心。

### 1.1 每日单词 (Daily Word)
*   **[x] 单词卡片展示**：正面英文与音标、发音播放，反面释义与中英双语例句（已完成基本组件）。
*   **[x] 释义选择题测试**：给出英文选择中文，或给出中文选择英文（已完成）。
*   **[ ] 拼写小测**：根据释义或发音拼写出完整单词（待开发，优先级：高）。
*   **[x] 收藏生词**：在背诵过程中随时点击收藏标记到生词本（已完成）。

### 1.2 每日句子 (Daily Sentence)
*   **[x] 句子重排（连词成句）**：打乱句中单词顺序，用户手动拖拽/点击重排（已完成）。
*   **[x] 句子翻译测试**：中译英或英译中选择题（已完成）。
*   **[x] 遮挡填空**：隐藏句子中的部分关键介词/语法词，用户拼写或选择填空（已完成）。
*   **[ ] 语法分析展示**：对每日句子的主谓宾、从句结构进行树形或颜色高亮展示（待开发，优先级：中）。

### 1.3 每日听力 (Daily Listening)
*   **[x] 听力播放器**：支持播放、暂停、进度调节。
*   **[ ] 语速调节**：0.8x、1.0x、1.2x、1.5x 语速实时切换（待开发，优先级：高）。
*   **[ ] 听力单选测验**：盲听音频后完成配套的选择题（待开发，优先级：高）。
*   **[ ] 听写填空 (Dictation)**：听音频，在输入框中拼写出缺失的关键单词（待开发，优先级：中）。

### 1.4 每日文章 (Daily Article)
*   **[x] 双语切换**：一键切换“纯英文/中英对照”模式（已完成）。
*   **[x] 字体调节**：调节阅读页的字体大小（已完成）。
*   **[ ] 划词词典**：双击文章中的任意单词，直接弹窗查询释义并支持加入生词本（待开发，优先级：高）。
*   **[x] 阅读进度自动记录**：离开文章自动保存当前进度（已完成）。

## 📚 2. 单项专项训练模块 (Specialized Practice)

提供垂直领域的深挖练习。

### 2.1 音标专项 (Phonetics)
*   **[ ] 互动式音标表**：48个国际音标点击播放发音、展示口型图（待开发，优先级：中）。
*   **[ ] 发音对比**：录音功能，录制用户发音并与标准发音波形进行对照（待开发，优先级：低）。
*   **[ ] 听音辨音测试**：选出听到的正确音标（待开发，优先级：中）。

### 2.2 字词根专项 (Word Roots)
*   **[ ] 词根词缀库**：拉丁、希腊语高频词根分类展示（待开发，优先级：高）。
*   **[ ] 联想词汇树**：树形图展示相同词根下的派生词（待开发，优先级：中）。
*   **[ ] 拆词连线游戏**：将前缀、词根、后缀与正确含义连线（待开发，优先级：低）。

### 2.3 单词/句子/阅读/听力专项
*   **[ ] 垂直词库选择器**：用户可一键切换四六级、考研、雅思等词库（待开发，优先级：高）。
*   **[ ] 语法专题库**：包含定语从句、动名词等专题句型专项练习（待开发，优先级：中）。
*   **[ ] 精听练习**：逐句断点循环播放听力练习（待开发，优先级：中）。

---

## 🔄 3. 巩固复习与智能管理 (Consolidation & Review)

通过机制确保学习效果，防止遗忘。

### 3.1 错题集与生词本 (Smart Notebook)
*   **[x] 错题/生词自动收集**：将测试答错题、划词翻译的生词自动记入本地（已完成）。
*   **[ ] 消灭错题模式**：用户需要连续三次答对某道错题，该题才会被移出本列表（待开发，优先级：高）。
*   **[ ] 导出功能**：支持将生词本/错题集导出为 PDF 或 TXT（待开发，优先级：低）。

### 3.2 遗忘曲线复习引擎
*   **[ ] SM-2 记忆算法实现**：根据用户反馈对单词/句子的熟练度，自动安排 1/2/4/7/15 天的复习日程（待开发，优先级：高）。
*   **[ ] 智能复习任务生成**：每天的每日任务自动混入当天需复习的旧内容（待开发，优先级：高）。

## 🏆 4. 社交与动力系统 (Leaderboard & Gamification)

### 4.1 学习榜单 (Leaderboard)
*   **[ ] 积分引擎**：打卡、练习正确率转化为 Learning Points (LP)（待开发，优先级：高）。
*   **[x] 全球/好友排行榜**：周排名、日打卡天数排名（已完成基础 UI 与 mock 数据，待接入真实榜单 API）。
*   **[ ] 段位升级机制**：基于 LP 积分解锁不同学习称号（待开发，优先级：低）。

### 4.2 徽章与成就系统 (Achievements)
*   **[x] 成就系统首页**：展示所有徽章及其解锁条件（已完成）。
*   **[x] 基本成就计算**：连续打卡天数成就、背诵单词量成就（已完成基础功能，待持久化）。
*   **[ ] 分享海报生成**：解锁成就时一键分享（待开发，优先级：低）。

### 4.3 学习报告 (Study Reports)
*   **[x] 基础学习统计**：今日时长、今日背词（已完成）。
*   **[x] 折线图与环形图**：周学习时间折线图展示（已完成基础 UI，需绑定数据）。
*   **[x] 五维能力雷达图**：展示听说读写及词汇的雷达图分析（已完成，支持基于学习数据和打卡进度动态重绘）。

## 🤖 5. 智能与辅助工具 (Smart Utilities)

### 5.1 个性化阶段规划器 (Staged Planner)
*   **[ ] 测评定级**：初次登录进行 10 题简易测试以推荐初始学习难度（待开发，优先级：中）。
*   **[ ] 阶段性目标追踪**：完成当前阶段词汇/句子量后解锁下一阶段（待开发，优先级：高）。

### 5.2 AI 口语陪练 (AI Talking Coach)
*   **[x] 对话场景模拟**：模拟餐厅、购物等对话（已完成 UI 与 Mock 对答）。
*   **[ ] AI 语音应答**：对接大模型或语音合成，实现实时语音对话（待开发，优先级：低）。
*   **[ ] 口语表达纠错**：展示用户口语的语法错误及地道修改方案（待开发，优先级：低）。
